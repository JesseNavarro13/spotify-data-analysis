# spotify-data-analysis

## Overview
This project analyzes a dataset of Spotify track to identify patterns in song popularity and audio features.

## Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

## Analysis Performed
- Data cleaning
- Exploratory data analysis
- Feature correlation analysis
- Data visualization

## Key Insights
- Songs with higher danceability tend to have higher popularity.
- Energy shows moderate correlation with popularity.
- The distribution of song popularity is skewed towards lower popularity scores, with a long tail of highly popular songs.

## Dataset
Spotify Tracks Dataset (Kaggle)
